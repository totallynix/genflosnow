# Genflosnow Changelog
## [v1.1.0] Users Implementation - 8/2/26
- Introduction of User accounts in addition to the existing Admin accounts
- Currently has no functionality, features will be rolled out in subsequent updates
- Guests can sign up for User accounts via /login
- Username will be shown on homepage

## [v1.0.0] Github First Release - 8/2/26
- Imported index into Github for all existing sites due to loss of local files during device formatting
  - Root/Homepage
  - Login
  - Admin
  - Quiz
  - Create
  - Edit
- Existing features include:
  - Working quiz feature with quiz title, description, questions with 4 options each: red triangle, blue diamond, green square, yellow circle, and a timer feature; very similar to Kahoot
  - Admin interface to create new and edit existing quizzes; Admins could amend all the variables mentioned above
  - Quizzes had the "Hidden" feature to only allow players with the direct link to access the quiz, and you could also open or close quizzes so players can or cannot play the quiz
  - Scoring system with streaks adding on to a higher score
  - Total score shown at the top right of the screen and at the end of the game
  - Fullscreen functionality for compatible devices
